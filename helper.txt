#Autoactivate
Step 1. Create a .envrc file
#Inside your project:
echo 'source .venv/bin/activate' > .envrc

Step 2 install direnv
#Install direnv (Ubuntu 22.04/24.04):
sudo apt update
sudo apt install direnv -y 
OR sudo snap install direnv

Step 3. Hook direnv into your shell

Add this line at the bottom of your ~/.bashrc:
eval "$(direnv hook bash)"
